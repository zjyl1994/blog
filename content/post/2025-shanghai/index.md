# 豫园&城隍庙
豫园是典型的江南园林，门票要提前网上预约
![豫园](豫园.webp)
城隍庙不用门票，财神和月老那边人都络绎不绝，出于尊重没怎么拍神仙。这是外面小吃街的饭店。
![城隍庙](城隍庙.webp)

# 其他
很著名的抗战遗迹，致敬800壮士
![四行仓库](四行仓库.webp)

上海美术馆又名中华艺术宫，是当年世博会的中国馆，好多好多画。
![上海美术馆-鲁迅](上海美术馆-鲁迅.webp)
很多艺术画作，看懂的看不懂的，就这一张我很熟悉，因为他是上学时候教材里的鲁迅先生。

![朱家角](朱家角.webp)
江南水乡普通古镇，一条商业街沿河设置，后面全是在卖的小房子。老外超级多，韩语泰语还有东南亚口音的中文都有，看老外穿汉服打卡也挺有意思的。
另外，这的小杨生煎门店超小要等位。离地铁口也挺远，至少2km。

![地铁口](地铁口.webp)
很普通的城市一景，应该没人猜得到这是哪里吧？

![磁悬浮](磁悬浮.webp)
凭当日机票磁悬浮8折，40块钱，8分钟不到就到了浦东机场。

# 后记
上海元旦的温度和广东这边冬季差不多，没想象中的冷。
这下北上广深四个一线城市我算是都去过了。
